Alcohohol - BAC Estimator

What is that

This is my first Java work, a simple blood alcohol concentration estimator.
In a few of words, it Alcohohol allows the user to :
  1. Set simulation start-time and duration.
  2. Add some drink ( given alcohol % and quantity ).
  3. View a summary of what he drinked.
  4. View a graph of the BAC trend in his body.
